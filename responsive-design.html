Responsive Design

Door: r20222

18-01-2023

Op 18 januari kwam de maker van de browser Polypane, Kilian Valkhof, iets vertellen op school over responsive design. Hij gaf in zijn presentatie 10 belangrijke punten om aan te denken als je bezig bent met het maken van een website:

  • #0 viewport meta tag
  • #1 keep viewport simple
  • #2 mobile first
  • #3 design from content out
  • #4 use `em's` in mediaqueries
  • #5 min-width or max-width, pick one!
  • #6 avoid pixed dimensions
  • #7 use modern layout techniques
  • #8 leave room for text rendering
  • #9 decide in the browser

Na afloop van de powerpoint presentatie liet Kilian zijn gemaakte browser Polypane zien. Dit is een geweldige browser voor developers omdat deze heel veel verschillende ingebouwde functies bevat waarmee je gemakkelijk kan testen op verschillende punten waarmee je als developer rekening mee moet houden. Je kan zelfs testen op allerlei oogafwijkingen die mensen kunnen hebben zodat je je website goed accessible kan maken voor heel veel mensen.
